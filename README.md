# 循环&旋转&复制&混合 for MasterGo

一个面向 MasterGo 画布环境的矢量效率插件，围绕三个高频操作做了整理：

- `循环`
- `旋转复制`
- `混合`

目标不是做“大而全”的设计工具，而是把常用的重复、混合、复制类操作收进一个轻量插件里，减少手工重复编辑。

## 功能概览

### 1. 循环

对选中的单个图层做循环复制，并支持常见的位移和样式变化。

支持项：

- 迭代次数
- X / Y 位移
- 旋转步长
- 控制点
- 生成层级（在源图层上方或下方）
- 宽 / 高缩放步长
- 透明度起止值
- 填充起止色
- 描边起止色
- 描边粗细起止值
- 一键恢复默认参数

颜色输入支持浏览器原生取色器，也保留 Hex 输入行为。

### 2. 旋转复制

围绕选中图层做环形复制，用于做放射、圆周排列、节奏型图形。

支持项：

- 复制数量
- 半径
- 角度步长
- 起始角度
- 控制点
- 径向对齐开关
- 画布预览

### 3. 混合

对两个同父级图层生成中间层。

实现策略：

- 优先尝试路径混合
- 可提取轮廓时优先走 path morph
- 不可提取时回退到基础几何混合
- 支持颜色、透明度、尺寸、旋转等插值

## 使用方式

### 循环

1. 在画布中选中 1 个图层。
2. 切换到 `循环`。
3. 调整迭代次数、偏移、缩放、透明度和颜色参数。
4. 点击 `生成循环`。

### 旋转复制

1. 在画布中选中 1 个图层。
2. 切换到 `旋转复制`。
3. 设置复制数量、半径、角度步长和起始角度。
4. 如需预览，可打开预览开关。
5. 点击 `应用`。

### 混合

1. 在画布中选中 2 个同父级图层。
2. 切换到 `混合`。
3. 设置中间层数。
4. 点击 `创建`。

## 运行与构建

项目使用 Vite 构建，主线程与 UI 分离。

```bash
npm install
npm run build
```

开发模式：

```bash
npm run web
```

## 项目结构

- `lib/main.ts`：主线程入口，负责选区校验、消息分发、画布写入
- `lib/blend.ts`：混合插值和外观处理
- `lib/loopGenerator.ts`：循环复制生成逻辑
- `lib/copyRotateGenerator.ts`：旋转复制和预览逻辑
- `lib/pathBlend.ts`：路径混合
- `lib/guidedPathBlend.ts`：路径引导混合
- `lib/pathSource.ts`：路径提取
- `lib/svgPath.ts`：SVG 路径工具
- `messages/sender.ts`：UI 与主线程消息协议
- `ui/App.vue`：UI 入口
- `ui/pages/*`：各功能页面
- `ui/components/*`：通用表单和底部按钮组件
- `ui/composables/*`：各功能的状态与消息处理

## 说明

- 插件当前面向 MasterGo 的 `canvas` 场景。
- 混合、循环、旋转复制都依赖当前画布中的选区。
- 代码按“主线程 / 算法 / 消息协议 / UI”分层组织，便于后续继续加新功能。
